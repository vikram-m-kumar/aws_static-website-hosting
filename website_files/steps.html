Steps to Host Static Website

Home Steps
  1. Create an S3 bucket: Name it after your domain (e.g., yourdomain.com). Enable static website hosting and upload your HTML files.
  2. Make files public: Adjust bucket policy to allow public read access to your website content.
  3. Use AWS Certificate Manager (ACM): Request a free SSL certificate for HTTPS.
  4. Create a CloudFront distribution: Set your S3 website endpoint as the origin. Attach the SSL cert from ACM.
  5. Set up Route 53: Create a hosted zone and add an alias record pointing your domain to CloudFront.
